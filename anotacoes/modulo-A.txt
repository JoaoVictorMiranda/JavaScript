HTML-CSS-JavaScript a triade para criação de sites, uma não funciona sem a outra.
O javascript faz as interações entre itens dentro de um site sendo ir de um lugar ao outro ou simplesmente alterar alguma função com base em algum botão que o usuario apertou, ampliar uma imagem com base no mouse do cliente e escureçer o fundo.
JavaScript é de fato uma linguagem de programação diferente do HTML e CSS q são linguagens de marcação e de estilização.
O youtube é realmente completamente totalmente necessitado de JavaScript e o google é funcional sem javascript porem fica tudo antigo como se fosse em outra decada por isso o javascript é realmente muito necessario para todos os sites da internet pois
eles são baseados na triade dos sites (HTML-CSS-JavaScript)
